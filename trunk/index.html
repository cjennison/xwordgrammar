WHAT IS AN X-WORD? An X-Word is defined as the first word in a yes/no question.
There are 21 X-Words in the English language.

WHAT IS X-WORD GRAMMAR? X-Word Grammar is an editing method that helps authors of all ages to find and correct common english writing errors. This method is an easily learned system that helps to improve their writing.
